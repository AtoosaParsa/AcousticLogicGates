# AcousticLogicGates
Evolution of Acoustic Logic Gates in Granular Metamaterials
  
  This repository contains the code used for experiments presented in the following paper:

### Evolution of Acoustic Logic Gates in Granular Metamaterials
### Atoosa Parsa, Dong Wang, Corey S. O’Hern, Mark D. Shattuck, Rebecca Kramer-Bottiglio, and Josh Bongard
#### Applications of Evolutionary Computation – 25th International Conference, EvoApplications 2022
  
  
  
If using this code for academic purposes, please cite the paper above.
